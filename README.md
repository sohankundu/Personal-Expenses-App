# Personal Expenses App
 This is an app built with the Flutter framewok that is used to keep records of our expenses.
 We can add a transation by clicking the floating action button or by clicking the button on the right of the app bar. The home screen shows a chart and a list displaying all transactions. We can delete a transaction by clicking the trash bin icon.
